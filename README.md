📱 Google Play Store Sentiment Analysis

📌 Project Overview

This project performs Sentiment Analysis on user reviews from the Google Play Store. The aim is to analyze user feedback and classify sentiments (positive, negative, neutral) to gain insights into app performance and user satisfaction.

🎯 Objectives
Analyze user reviews from Play Store apps
Perform text preprocessing and cleaning
Apply Natural Language Processing (NLP) techniques
Classify sentiments of reviews
Visualize sentiment distribution
Generate insights for app improvement

🛠️ Technologies Used
Python 🐍
Pandas
NumPy
Matplotlib
Seaborn
NLTK / TextBlob / Scikit-learn
Jupyter Notebook

📊 Key Insights
Majority of reviews tend to be positive/negative (based on dataset)
Low-rated apps often have strong negative sentiment
Common complaints and praise keywords identified
User feedback highlights areas of improvement
